# Streamline admin users page layout

Update the admin users page to be more visually streamlined and less boxy.

## Changes

- **Remove section boxes**: The page currently has boxes/cards around each section. Drop those borders/containers so the page reads as a cleaner, more continuous layout.
- **Keep the header**: The header section with the student's name and email addresses is good as-is — leave it visually distinct.
- **Reposition action buttons**: Buttons for adding or revoking access currently sit at the bottom of each section. Move them to the right side of each section instead (right-aligned, alongside the section content rather than below it).
